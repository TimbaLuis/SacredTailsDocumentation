    public void SetGeometry()

    SetGeometryFromMesh(Mesh, UInt64, Boolean, Boolean, Boolean, AcousticTexture[], Single[], String)

    Convert the mesh into a geometry consisting of vertices, triangles, surfaces, acoustic textures and transmission loss values. Send it to Wwise with the rest of the AkGeometryParams to add or update a geometry in Spatial Audio. It is necessary to create at least one geometry instance for each geometry set that is to be used for diffraction and reflection simulation. See SetGeometryInstance().

    Declaration
    public static void SetGeometryFromMesh(Mesh mesh, ulong geometryID, bool enableDiffraction, bool enableDiffractionOnBoundaryEdges, bool enableTriangles, AcousticTexture[] acousticTextures = null, float[] transmissionLossValues = null, string name = "")
    Parameters
    Type Name Description
    UnityEngine.Mesh mesh

    The mesh representing the geometry to be used for diffraction and reflection simulation.
    UInt64 geometryID

    A unique ID representing the geometry.
    Boolean enableDiffraction

    Enable the edges of this geometry to become diffraction edges.
    Boolean enableDiffractionOnBoundaryEdges

    Enable the boundary edges of this geometry to become diffraction edges. Boundary edges are edges that are connected to only one triangle.
    Boolean enableTriangles

    When enabled, the geometry triangles are indexed for ray computation and used to computed reflection and diffraction. Set EnableTriangles to false when using a geometry set only to describe a room, and not for reflection and diffraction calculation.
    AK.Wwise.AcousticTexture[] acousticTextures

    The acoustic texture of each surface of the geometry. Acoustic textures describe the filtering when sound reflects on the surface.
    Single[] transmissionLossValues

    The transmission loss value of each surface of the geometry. Transmission loss is the filtering when the sound goes through the surface.
    String name

    A name for the geometry.

    SetGeometryInstance(UInt64, UInt64, UInt64, Transform)

    Add or update an instance of the geometry by sending the transform of this component to Wwise. A geometry instance is a unique instance of a geometry set with a specified transform (position, rotation and scale). It is necessary to create at least one geometry instance for each geometry set that is to be used for diffraction and reflection simulation.

    Declaration
    public static void SetGeometryInstance(ulong geometryInstanceID, ulong geometryID, ulong associatedRoomID, Transform transform)
    Parameters
    Type Name Description
    UInt64 geometryInstanceID

    A unique ID to for the geometry instance. It must be unique amongst all geometry instances, including geometry instances referencing different geometries.
    UInt64 geometryID

    The ID of the geometry referenced by this instance.
    UInt64 associatedRoomID

    The ID of the room this geometry is encompassed in, if any.
    UnityEngine.Transform transform

    The transform to be applied to the geometry to convert it in world positions.
